## Agents

Agents are specialist personas designed for specific roles.

| Agent | Description |
|-------|-------------|
| **Frontend** | Expert in React, Vue, CSS, and modern UI capabilities. |
| **Backend** | Specialist in API design, databases, and server logic. |
| **DevOps** | Infrastructure as Code, CI/CD, and deployment. |
| **Architect** | High-level system design and patterns. |
| **QA** | Testing strategies and bug hunting. |

## Skills

Skills are modular capabilities that give agents specific knowledge or tools.

- **Docker**: Container management
- **Git**: Version control operations
- **SQL**: Database querying
- **React**: React specific patterns
- **Go**: Golang best practices

## Workflows

Workflows are predefined sequences of actions.

- **Release**: Automate release process
- **Refactor**: Safe refactoring steps
- **Audit**: Codebase auditing
